## Getting Started

### Prerequisites

- Ruby version ~> 2.6
- IDE (if you want to test)

### Dependencies

- Rubocop

### Install

`gem rubocop` `bundle install`

### Run tests

Rubocop lint
`rubocop .`

### Deployment

`ruby enumerable.rb`
